# 🩺 AI-Based Medicine Reminder and Health Tracker System

An end-to-end Machine Learning application designed to bridge predictive health analytics with real-time, user-facing medical intervention. This ecosystem ingests multi-feature patient vitals to generate dynamic health assessments and smart medication alerts.

## 🚀 Core Features
* **Dual-Engine Predictive Pipeline:** Implements multi-target prediction architectures solving both regression and classification challenges simultaneously.
* **Continuous Health Analytics:** Uses a Linear Regression model to calculate a precise, transparent "Overall Health Index Rating" (0-100) based on user lifestyle features.
* **Automated Medication Despatch:** Leverages a Decision Tree Classifier to build clinical logic flowcharts, categorizing risk levels to trigger instant patient notifications.
* **State-Persistent Multi-Tab UI:** Designed using Streamlit, separating data collection components from the client-facing clinical emergency notification center.

## 🛠️ Tech Stack & Architecture
* **Frontend UI Framework:** Streamlit
* **Data Engineering & Manipulation:** Pandas, NumPy
* **Machine Learning Pipelines:** Scikit-Learn (LinearRegression, DecisionTreeClassifier)
* **Data Scaling & Encoding:** StandardScaler, LabelEncoder

## 📊 Feature Telemetry Tracked
1. Sleep Duration (Hours)
2. Water Intake Volume (Liters)
3. Daily Active Physical Exercise (Minutes)
4. Recorded Heart Rate Metric (BPM)
5. Calculated Daily Active Metabolism (Kcal)
6. Patient Demographic Age Profile
